# Omarchy Liquid Crystal Theme

Un tema para Omarchy (Hyprland) inspirado en la estética Liquid Crystal de Apple.

## Preview
![Preview](preview.png)

## Instalación

### Método Recomendado (Bash)
Puedes instalar este tema directamente usando el comando `omarchy-theme-install` si tienes Omarchy instalado:

```bash
omarchy-theme-install https://github.com/k4ditano/liquidclass-omarchy.git
```

### Instalación Manual
Copia los archivos de configuración a sus respectivas ubicaciones en `~/.config/`.

- `hyprland.conf` -> `~/.config/hypr/hyprland.conf`
- `waybar.css` -> `~/.config/waybar/style.css`
- `alacritty.toml` -> `~/.config/alacritty/alacritty.toml`
- `kitty.conf` -> `~/.config/kitty/kitty.conf`
- `mako.ini` -> `~/.config/mako/config`
- `walker.css` -> `~/.config/walker/style.css`
- `btop.theme` -> `~/.config/btop/themes/liquid-crystal.theme`
- `neovim.lua` -> `~/.config/nvim/init.lua` (o intégralo en tu config)

## Características
- **Hyprland:** Bordes redondeados, desenfoque (blur), sombras suaves y animaciones fluidas.
- **Waybar:** Barra estilo cristal oscuro con botones brillantes.
- **Terminal:** Temas para Alacritty y Kitty con fondo transparente.
- **Colores:** Tonos cian, azul y transparencias oscuras.

## Requisitos
- Hyprland
- Waybar
- Alacritty / Kitty
- Mako
- Walker (opcional, lanzador)
- Fuentes: SF Pro Display (recomendada) o Roboto, JetBrainsMono Nerd Font.
